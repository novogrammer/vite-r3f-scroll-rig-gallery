# Vite r3f-scroll-rig Gallery

全画面Canvasでr3fとHTMLと連動させる

demo https://novogrammer.github.io/vite-r3f-scroll-rig-gallery/


派生元 https://github.com/novogrammer/vite-r3f-gallery

## インストール
```
npm install
```

## 開発
```
npm run dev
```

## ビルド
```
npm run build
```

## ビルドをプレビュー
```
npm run preview
```
